
# Entrata Training Assignment

## 📌 Project Overview
This project is a full-stack e-commerce website built using PHP, MySQL, HTML, CSS, and JavaScript. It demonstrates end-to-end development, testing, containerization, and deployment on AWS.

---

## 🎯 Phase 0: Training (Foundation)

Completed comprehensive training covering core technologies with hands-on code practice and scripts.

### 📚 Topics Covered:
- **Linux**
  - File system, navigation, permissions, process management
  - Shell scripting and automation basics

- **Git & GitHub**
  - Version control, branching, merging
  - Collaboration workflow and best practices

- **JavaScript**
  - Variables, operators, loops, functions
  - DOM manipulation, events, validation

- **PHP**
  - Syntax, data types, functions, arrays
  - Forms handling, sessions & cookies
  - File handling, APIs, namespaces, OOP basics

- **Docker**
  - Docker commands, Dockerfile, Docker Compose
  - Networking, volumes, debugging, best practices

### 💻 Practice Work:
- Created multiple scripts for Linux, Git, and Docker
- Built mini examples for PHP and JavaScript
- Hands-on command-line and DevOps workflows
