# Entrata Training Assignment

## 📌 Project Overview

This project is a full-stack e-commerce website built using **PHP, MySQL, HTML, CSS, and JavaScript**.

---

## 🚀 Phase 1: Development

* Developed a complete clothing e-commerce website
* Features:

  * User authentication
  * Product listing
  * Cart system
  * Checkout system
  * Admin panel

---

## 🧪 Phase 2: Testing

* Implemented unit testing using PHPUnit
* Tested:

  * Login functionality
  * Checkout logic
  * Input validation

---

## 🐳 Phase 3: Dockerization

* Containerized application using Docker
* Services:

  * PHP + Apache
  * MySQL
* Used docker-compose for orchestration

---

## ☁️ Phase 4: AWS Deployment

* Deployed Dockerized application on AWS EC2
* Configured:

  * Security groups
  * Public IP access
* Application is live at:

👉 http://3.111.225.52:8080/:8080
